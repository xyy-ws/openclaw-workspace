# 本次项目笔记（PRD 风格）

## 项目
- 名称：Habit MVP（OpenClaw）
- 目标：快速生成可用可测可预览的习惯养成 Web 应用
- 范围：MVP 功能闭环 + 域名预览 + 高级感 UI 首轮

## 用户故事（节选）
### US-001 需求澄清与范围定义
- 验收标准：
  - 明确平台/登录/提醒/备份策略
  - 明确 MVP 边界与成功标准
- 状态：通过

### US-002 功能闭环实现
- 验收标准：
  - API：设备、习惯、打卡、同步接口可用
  - Web：核心页面与流程可操作
  - 测试：单测/构建/E2E 通过
- 状态：通过

### US-003 可访问预览部署
- 验收标准：
  - 域名路径预览可访问
  - 子路径路由无 404
  - 遇到进程掉线可恢复
- 状态：通过

### US-004 视觉升级与交互增强
- 验收标准：
  - 现代高级感视觉落地
  - hover/鼠标交互/滚动入场可用
  - 兼容 reduced motion
- 状态：通过

### US-005 过程资产沉淀
- 验收标准：
  - 复盘文档产出
  - 标准模板产出
  - 下次可直接复用
- 状态：通过
